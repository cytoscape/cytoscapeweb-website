NodeShape

This object represents node shapes. In actuality, it is just a string.

Possible values:

  • ELLIPSE
  • RECTANGLE
  • TRIANGLE
  • DIAMOND
  • HEXAGON
  • OCTAGON
  • PARALLELOGRAM
  • ROUNDRECT
  • VEE

ArrowShape

This object represents edge arrow shapes. In actuality, it is just a string.

Its value must be one of:

  • NONE
  • DELTA
  • ARROW
  • DIAMOND
  • CIRCLE
  • T
